# FleetDash

A high-performance fleet telemetry dashboard built with the MERN stack for real-time vehicle tracking, geofence alerts, and scalable data visualization.

## 🚀 Project Overview

FleetDash is a real-time fleet management application that allows logistics companies to monitor thousands of vehicles, track their locations, and receive instant geofence alerts.

## 🛠️ Tech Stack

- Frontend: React
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-Time Communication: Socket.io
- Caching: Redis
